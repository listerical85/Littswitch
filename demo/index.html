Littswitch

Littswitch, An easy to use jQuery plugin for making iOS 7 like toggles. It requires jQueryUI and is very easy to setup, just add the js/less|css files, add 3 custom data attributes and you're ready to go.

Getting started

  1. Download the latest version of Littswitch.
  2. Make sure you have jQuery and jQueryUI loaded in your site. (If not, they are included on Github). <script src="js/jquery-1.10.2.min.js"></script>
    <script src="js/jqueryui.js"></script>
  3. Find littswitch.js in the js folder and include it in your site, after you have loaded jQuery <script src="js/littswitch.js"></script>
  4. In the css folder you will find littswitch.css include it in your site. <link rel="stylesheet" href="css/littswitch.css">
  5. Now you're ready to use Littswitch, add these 3 data attributes to your checkbox, data-littswitch, data-littswitch-on and data-littswitch-off. See the example below.
    6. <input type="checkbox" data-littswitch="true" data-littswitch-on="On" data-littswitch-off="Off"/>
  6. The data-littswitch makes sure littswitch will replace this checkbox, data-littswitch-on and data-littswitch-off, contain the values for the on and off states.
  7. Enjoy Littswitch!
